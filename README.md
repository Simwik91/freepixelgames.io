# Free Pixel Games

This project, `freepixelgames.io`, is a collection of free pixel art games designed to be played directly in your browser. It aims to provide classic-style gaming experiences without the need for downloads or sign-ups.

## Features

-   **Browser-Based Games**: Play directly in your web browser.
-   **Pixel Art Style**: Enjoy retro-inspired graphics.
-   **Free to Play**: No costs or subscriptions.
-   **Privacy-Focused**: Games run locally in your browser; no user data is collected or stored on servers.

## Available Games

-   **Snake**: A classic arcade game where you control a snake trying to eat food and grow longer, avoiding collisions with walls and its own tail.

## Usage

1.  Open `index.html` in a web browser.
2.  Navigate to the desired game from the list (e.g., "Snake").
3.  Enjoy playing!
